Conteúdo da Disciplina

Ementa

Estudo das principais estruturas de dados e sua implementação utilizando a linguagem C. Aborda desde conceitos básicos como vetores e matrizes até estruturas mais complexas como listas encadeadas, pilhas, filas e filas circulares. Inclui o trabalho com structs, ponteiros, alocação dinâmica de memória e funções, proporcionando uma base sólida para o desenvolvimento de algoritmos eficientes e a resolução de problemas computacionais.

Tópicos Abordados

  • Revisão de conceitos básicos da linguagem C
  • Trabalho com vetores e matrizes
  • Structs e tipos de dados compostos
  • Ponteiros e alocação dinâmica de memória
  • Funções e modularização de código
  • Listas encadeadas (simples e duplamente encadeadas)
  • Pilhas (LIFO) e suas operações básicas
  • Filas (FIFO) e filas circulares
  • Implementação prática de estruturas de dados
  • Análise de complexidade de algoritmos

Metodologia

A disciplina é ministrada de forma prática, com aulas expositivas seguidas de exercícios de implementação em linguagem C. Os alunos desenvolvem programas que aplicam os conceitos teóricos, desde estruturas simples até as mais complexas. São realizadas atividades individuais e em grupo, com projetos que simulam problemas reais de programação. A avaliação inclui exercícios práticos, implementação de estruturas de dados, provas teóricas e análise de código produzido. O ambiente de desenvolvimento utilizado é o Code::Blocks ou Dev-C++, com ênfase na depuração e boas práticas de programação.

Aprendizados

Até o momento, a disciplina proporcionou uma compreensão sólida de como organizar e manipular dados de forma eficiente na memória. Aprendi a trabalhar com ponteiros e alocação dinâmica, conceitos fundamentais para estruturas mais complexas. A implementação de listas encadeadas me mostrou como criar estruturas flexíveis que crescem conforme a necessidade, diferente dos vetores estáticos. As pilhas e filas, por sua vez, demonstraram como a escolha da estrutura de dados adequada pode simplificar a solução de problemas específicos. Estou desenvolvendo uma mentalidade mais analítica para escolher a estrutura mais adequada para cada situação, considerando fatores como eficiência e consumo de memória.
