Conteúdo da Disciplina

Ementa

Estudo de técnicas avançadas de modelagem, implementação e otimização de bancos de dados relacionais e não relacionais. Abordagem prática com estudos de caso e projetos reais.

Tópicos Abordados

  • Modelagem de dados avançada
  • Otimização de consultas SQL
  • Bancos de dados NoSQL (MongoDB, Cassandra)
  • Data Warehousing e Business Intelligence
  • Big Data e processamento distribuído
  • Segurança em bancos de dados
  • Replicação e alta disponibilidade
  • Bancos de dados em memória
  • Integração de sistemas heterogêneos

Metodologia

A disciplina combina aulas teóricas com práticas em laboratório, utilizando sistemas de gerenciamento de banco de dados como PostgreSQL, MySQL e MongoDB. Os alunos desenvolvem projetos práticos que simulam cenários reais de implementação e otimização de bancos de dados.

Aprendizados

Esta disciplina proporciona uma visão abrangente sobre os desafios atuais no gerenciamento de dados em larga escala, preparando os alunos para tomar decisões estratégicas sobre armazenamento, processamento e análise de dados em ambientes corporativos complexos.
